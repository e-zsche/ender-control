# Ender Control

Controls an Ender-3 filament printer via a USB-serial connection. Depends on the wonderful gscrib library to
process and send GCode commands.

gscrib:
- [https://github.com/joansalasoler/gscrib](source)
- [https://gscrib.readthedocs.io/en/latest/](docs)

# Installation

Put the library in your project directory and make sure to have gscrib installed in your python environment.
